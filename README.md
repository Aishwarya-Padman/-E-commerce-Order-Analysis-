# -E-commerce-Order-Analysis-
This project analyzes e-commerce order data using PostgreSQL. It demonstrates database creation, table design, and writing advanced SQL queries for business insights. The project covers:

✨ Project Highlights:

🗂️ Database Design: Multiple related tables (Orders, Customers, Products)
🧹 Data Cleaning & Insertion: Structured and cleaned data for analysis
🔗 Complex Queries: Using joins, aggregations, subqueries, and window functions
💰 Business Insights: Analyze total revenue, average order value, top-selling products, and top customers
⚡ Query Optimization: Faster execution and accurate reports
📊 Reporting: Generate reports to support business decision-making

🛠️ Tools Used: PostgreSQL, pgAdmin

📌 Key SQL Concepts Practiced:

🔗 Joins: INNER, LEFT, RIGHT, FULL
📊 Aggregations: GROUP BY, HAVING
🎯 Window Functions: ROW_NUMBER, RANK, DENSE_RANK
🕵️ Subqueries & Correlated Subqueries
⚡ Query Optimization
